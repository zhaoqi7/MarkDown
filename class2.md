linux

显示当前文件夹  
>**6.Cd是什么命令**

输出

>**20.直接从工作区提交到仓库的命令**    
git commit -a -m  
>**21.显示变更信息的命令**    
git  status  
>**22.查看历史信息的命令**       
git log  
>**23.Commit的意义是**      
提交信息

>**25.Push的意义是**
